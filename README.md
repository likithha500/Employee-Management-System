# Employee Management System

## Overview

The Employee Management System is a desktop-based application developed using Java Swing and MySQL to simplify the management of employee records within an organization. The application provides an intuitive graphical user interface for performing various employee management operations efficiently.

## Features

* Add new employee records
* Update existing employee details
* Delete employee records
* Search and view employee information
* Manage employee details such as:

  * Employee ID
  * Name
  * Department
  * Designation
  * Salary
  * Joining Date
* Store and retrieve data using MySQL database

## Tech Stack

* **Programming Language:** Java
* **GUI Framework:** Java Swing
* **Database:** MySQL
* **Database Connectivity:** JDBC

## Project Structure

* **UI Layer:** Handles the graphical user interface using Java Swing.
* **Business Logic Layer:** Processes employee-related operations.
* **Database Layer:** Manages database connectivity and CRUD operations using JDBC.

## Getting Started

1. Clone the repository.
2. Create the required MySQL database and tables.
3. Update database credentials in the project configuration.
4. Run the application using your preferred Java IDE.
